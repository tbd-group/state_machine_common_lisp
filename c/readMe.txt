This directory includes an example of State Machine (SM) implementation in C.

How to build?
    mkdir build     # only if build directory does not exist
    cd build
    cmake ..        # only if Makefile does not exist or needs an update 
    make

How to run?
    build/smEngine
    The program responses to key '1', '2', ...,'6'

How to exit?
    Press Esc

For more details see the project's Confluence page
    http://adnconfluence.corp.amazon.com/display/PA/SM+C
